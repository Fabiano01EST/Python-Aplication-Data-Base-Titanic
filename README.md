# Python Data Analysis: Titanic Dataset

## Descrição do Projeto

Este projeto tem como objetivo realizar uma análise de dados do famoso conjunto de dados Titanic. O dataset contém informações sobre os passageiros do navio Titanic, incluindo características como idade, sexo, classe social, tarifa paga, entre outros. A análise busca explorar os dados para identificar padrões e insights, como fatores que podem ter influenciado a sobrevivência dos passageiros.

### Estrutura do Projeto

- **aplicação_pyhton.html**: Arquivo HTML que pode conter visualizações ou relatórios gerados a partir da análise dos dados.
- **aplicação_pyhton.ipynb**: Notebook Jupyter contendo o código Python utilizado para a análise dos dados do Titanic.
- **titanic3.csv**: Arquivo CSV com o conjunto de dados do Titanic, que será utilizado como base para a análise.
- **R/**: Diretório que pode conter scripts ou arquivos relacionados à análise de dados em R.
  - **Python/**: Subdiretório que pode conter scripts Python adicionais ou complementares ao projeto.

### Funcionalidades

1. Carregamento e limpeza dos dados do Titanic.
2. Análise exploratória dos dados (EDA - Exploratory Data Analysis).
3. Visualização de dados utilizando bibliotecas como Matplotlib e Seaborn.
4. Identificação de padrões e fatores que influenciam a sobrevivência dos passageiros.
5. Geração de relatórios e gráficos para apresentação dos resultados.

### Tecnologias Utilizadas

- **Python**: Linguagem principal utilizada para a análise de dados.
- **Jupyter Notebook**: Ferramenta para desenvolvimento e execução do código.
- **Bibliotecas Python**:
  - Pandas: Manipulação e análise de dados.
  - NumPy: Operações matemáticas e manipulação de arrays.
  - Matplotlib e Seaborn: Visualização de dados.

### Como Executar o Projeto

1. Certifique-se de ter o Python instalado em sua máquina.
2. Instale as dependências necessárias utilizando o comando:
   ```
   pip install -r requirements.txt
   ```
3. Abra o arquivo `aplicação_pyhton.ipynb` em um ambiente Jupyter Notebook.
4. Execute as células do notebook para realizar a análise dos dados.
